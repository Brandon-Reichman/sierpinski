# Sierpiński

A [vuejs](https://vuejs.org/) component that draws the [Sierpiński triangle](https://en.wikipedia.org/wiki/Sierpi%C5%84ski_triangle)

### Example

https://sierpinski.surge.sh/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
